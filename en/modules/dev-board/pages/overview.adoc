== Appearance
The appearance of the Dev Board is shown below.

image::Appearance.svg[Appearance.svg]

[width="100%",cols="1,2,6",options="header",]
|===
|Number |Part Name |Description
|1 |Main Power Connector |Connects the AC adapter.
|2 |Power Switch |Turns the main power ON/OFF.
|3 |Power LED Indicator |Indicates the output status of various power supplies.
|4 |Power Selector |Switches the input source for the main power and UIO power.
|5 |OBC |Allows mounting of the SC-OBC Module A1.
|6 |User Switch |DIP switch allowing users to freely configure functions.
|7 |User LED |Yellow-green LED available for user-defined use.
|8 |CAN Connector |Terminal block for connecting CAN communication cables.
|9 |USB Connector |Connects a USB Micro-B plug. Usable for FPGA JTAG, CPU JTAG, FPGA UART (Console), and TRCH UART.
|10 |User Temperature Sensor |A temperature sensor freely usable by the user.
|11 |PICkit Connector |Connects a Microchip PICkit.
|12 |Universal Area |Universal Area
|13 |USB Connector |Allows connection of a USB Type-A plug on the front and a Micro-B plug on the back. Can be used for SC Bus communication.
|14 |UIO Power Connector |A terminal block allowing the user to supply UIO power at any desired voltage.
|15 |Main Power Connector |A terminal block allowing the user to supply main power using a source other than the AC adapter.
|===
